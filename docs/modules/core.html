getCurvePoints

  • getCurvePoints(points: Array<any>, tension?: number, numOfSeg?: number, close?: boolean): Float32Array
    • Defined in Core/Core.ts:446

    Calculates an array containing points representing a cardinal spline through given point array. Points must be arranged as: [x1, y1, x2, y2, ..., xn, yn].

    There must be a minimum of two points in the input array but the function is only useful where there are three points or more.

    The points for the cardinal spline are returned as a new array.

    Parameters

    • points: Array<any>
    • Default value tension: number = 0.5
    • Default value numOfSeg: number = 25
    • Default value close: boolean = false

    Returns Float32Array
